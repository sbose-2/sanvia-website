# SANVIA Static Website Bundle

This is a static, single-page site for SANVIA designed for fast handoff.

## How to Use
1. Place a looping background video at `assets/sanvia-bg.mp4` (optional).
2. Open `index.html` in any modern browser, or host the folder with any static host (S3, Netlify, GitHub Pages, etc.).

## Stack
- Tailwind CSS via CDN
- No build step required
- Responsive and accessible
